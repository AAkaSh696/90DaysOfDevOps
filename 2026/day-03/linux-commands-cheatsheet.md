## Process Management 
- ps: static snapshot of running  processes in current terminal
- ps-aux: detailed snapshot of all running processes on the system
- top: dynamic,real time view of running processes and system usage
- htop: similar to top but provides user friendly interface(modern way)
- pstree: show parent child relation tree of process
- kill: used to terminate process 
## File System 
- df -h: shows disk usage( in human readable)
- cd: to change directory
- mkdir: to make new directory
- touch: to make new file
- ls: to list contents
- pwd: to print current working directory
- rm: to remove files
- rm -r: to remove recursively (directory/files)
- mv: to move or rename file or directory
- find: search content of files and directories
- free: show RAM & swap memory information
## Networking Troubleshooting 
- ip addr: view and manage network interfaces addresses
- ping: check network connectivity btw own system and another system(server,website or ip address)
- tracepath: gives path that packets take from src to des
- wget: to download files from internet
- hostname: display or set the system's identifier on a network
- whois: provides domain name information
